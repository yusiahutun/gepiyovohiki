PES 2020 Patch Option File: How to download PS4, PC option files to get official licences, kits, bad

Publish date: 2024-06-19

The PES 2020 option file (otherwise known as the PES 2020 patch) is one of the most heavily sought after downloads each year as Konami release their annual release.

This year FIFA 20’s top competitor has upped its game in a bid for football video game supremacy.

Konami recently announced that they had struck an exclusive deal to bring with Juventus to the game at EA Sports expense.

However, once again if you were hoping to play with the official Premier League kits such as Chelsea and Manchester City, you will be disappointed.

Unsurprisingly for some players, the lack of official clubs is what makes them pick FIFA every year.

But fear not, we have a solution for you.

There is a feature in PES which makes it possible to download on option file which adds the league names, team names and kits which are missing on the game.

While having the default names doesn’t impact the action, there is no doubt having real names will make it feel more genuine.

Below is a step-by-step guide to how to download the PES 2020 option files.

How to download PES 2020 option file?

PES option files are only available for PS4 and PC, so Xbox players will sadly be missing out.

First you need to download the PES 2020 option file, which can be found on the PES patch or PES World websites. Other places, such as PES Universe will also have the option files, but are behind a paywall.

However, keep in mind PES World are not releasing a Day 1 Option File.

Instead, they will be releasing it on, or shortly after September 12.

They explained: “This is so that we can ensure there are no problems with file after the live update”.

How to download PES 2020 option file on PS4?

Once they are available, download the file and put the files in a folder named WEPES on a USB device, which you plug into your console.

Now head over to the Extras tab, followed by Edit.

Scroll down to Import/Expert and accept the Data Import Precautions.

Head to Import Competition and select the USB storage device, and you should see all the files.

You can either select each individual one, or select all by pressing square.

Confirm this, and ignore the Advanced Settings by pressing OK. Once you have done this you can repeat the process for Import Team.

Press Exhibition and you can see all the official crests and kits.

How to download PES 2020 option file on PC?

Once available, download the option file (.rar file) and extract it using a file extractor program such as 7-Zip.

You then need to extract the option file into the same place as the game is installed.

For example, if you are a Steam player, this would normally be C:\Steam\SteamApps\Common\Pro Evolution Soccer 2020.

You can check if the installation has worked in the Edit menu in the game.

Then head to the Teams menu and check the names, logos and kits.
